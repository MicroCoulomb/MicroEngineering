## MicroPrelegal
A platform for drafting prelegal documents. 

## Status
This project is currently in progress and is expected to be completed within a week.

## Run
The current foundation runs from Docker and serves the app at `http://localhost:8000`.

Windows:
`powershell -ExecutionPolicy Bypass -File .\scripts\start-windows.ps1`
`powershell -ExecutionPolicy Bypass -File .\scripts\stop-windows.ps1`

Linux:
`bash ./scripts/start-linux.sh`
`bash ./scripts/stop-linux.sh`

macOS:
`bash ./scripts/start-mac.sh`
`bash ./scripts/stop-mac.sh`
